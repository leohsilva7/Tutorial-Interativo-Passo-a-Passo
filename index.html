Como usar um ipconfig no Terminal

Progresso do tutorial:

Passo 1: Abrir o Prompt de Comando
  • Usando o atalho "Executar": Pressione as teclas Windows + R para abrir a janela "Executar". Digite cmd e pressione Enter.
Passo 2: Pesquisando
  • Pesquisando no Menu Iniciar: Clique no Menu Iniciar (ou na barra de pesquisa) e digite cmd (para o Prompt de Comando) ou powershell (para o PowerShell) e pressione Enter.
Passo 3: Rode o comando ipconfig
  • Com o terminal aberto, digite o seguinte comando: 
    ipconfig
Passo 4: Analise o resultado
  • O terminal irá mostrar o endereço IP e outras informações. O endereço IPv4 é a variável que representa o endereço IP do seu computador na rede local.
